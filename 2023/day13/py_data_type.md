Integer (int): a whole number, e.g. 1, 2, -5
Float: a decimal number, e.g. 3.14, -0.01
String (str): a sequence of characters, e.g. "hello", "goodbye"
Boolean (bool): a value that can be either True or False
List: an ordered collection of items, e.g. [1, 2, 3]
Tuple: an ordered collection of items that cannot be modified, e.g. (1, 2, 3)
Set: an unordered collection of unique items, e.g. {1, 2, 3}
Dictionary: a collection of key-value pairs, e.g. {"a": 1, "b": 2}
Python also supports other types such as complex numbers, bytes, and date time objects.
